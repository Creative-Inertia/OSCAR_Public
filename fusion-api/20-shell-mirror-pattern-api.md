## Shell Features

Shell features hollow out solid bodies by removing one or more faces and applying a uniform wall thickness to the remaining geometry.

### ShellFeatures Collection

Access via `rootComp.features.shellFeatures`

#### Methods

| Method | Returns | Description |
|--------|---------|-------------|
| `createInput(inputEntities, isTangentChain)` | ShellFeatureInput | Creates an input object for defining shell parameters before creation |
| `add(input)` | ShellFeature | Creates a new shell feature from a ShellFeatureInput |
| `item(index)` | ShellFeature | Returns the shell feature at the given index |
| `itemByName(name)` | ShellFeature | Returns the shell feature with the given name |

#### Properties

| Property | Type | Description |
|----------|------|-------------|
| `count` | Integer | Number of shell features in the collection |

### ShellFeatures.createInput()

```python
shellFeatureInput = shellFeats.createInput(inputEntities, isTangentChain)
```

| Parameter | Type | Description |
|-----------|------|-------------|
| `inputEntities` | ObjectCollection | Collection of faces to remove and/or bodies to shell |
| `isTangentChain` | Boolean | Whether tangentially connected faces are automatically included. Defaults to `True` |

Returns a `ShellFeatureInput` object (or `None` on failure).

### ShellFeatureInput - Properties

| Property | Type | R/W | Description |
|----------|------|-----|-------------|
| `inputEntities` | ObjectCollection | R/W | Faces to remove and/or bodies to shell. Cannot mix face and body inputs. Tangent-connected faces auto-included when `isTangentChain` is True |
| `insideThickness` | ValueInput | R/W | Inside (inward) shell thickness |
| `outsideThickness` | ValueInput | R/W | Outside (outward) shell thickness |
| `isTangentChain` | Boolean | R/W | Whether tangentially adjacent faces are automatically included. Defaults to True |
| `shellType` | ShellTypes | R/W | Shell algorithm type. Defaults to `SharpOffsetShellType` |
| `creationOccurrence` | Occurrence | R/W | Target occurrence when referencing geometry from another component |
| `targetBaseFeature` | BaseFeature | R/W | Base feature to associate with (requires startEdit/finishEdit) |

### ShellFeature Object - Properties

| Property | Type | R/W | Description |
|----------|------|-----|-------------|
| `name` | String | R/W | Feature name in browser/timeline |
| `bodies` | BRepBodies | R | Bodies modified or created by feature |
| `faces` | BRepFaces | R | Faces generated by the shell feature |
| `inputEntities` | ObjectCollection | R | Input faces/bodies |
| `insideThickness` | ModelParameter | R | Inner shell thickness parameter |
| `outsideThickness` | ModelParameter | R | Outer shell thickness parameter |
| `shellType` | ShellTypes | R/W | Shell algorithm type |
| `isTangentChain` | Boolean | R/W | Tangent face inclusion behavior |
| `isSuppressed` | Boolean | R/W | Whether feature is suppressed |
| `isParametric` | Boolean | R | Whether in parametric design |
| `healthState` | FeatureHealthStates | R | Feature health |
| `errorOrWarningMessage` | String | R | Error/warning message if unhealthy |
| `timelineObject` | TimelineObject | R | Timeline entry |
| `parentComponent` | Component | R | Parent component |

### ShellFeature Object - Methods

| Method | Description |
|--------|-------------|
| `deleteMe()` | Delete the feature |
| `dissolve()` | Remove feature, keep only geometry (non-parametric only) |
| `setInputEntities(inputEntities)` | Set faces to remove and bodies to shell |
| `setThicknesses(insideThickness, outsideThickness)` | Configure inside and outside thickness values |

### ShellTypes Enum

| Value | Description |
|-------|-------------|
| `SharpOffsetShellType` | Use sharp corners when creating the shell (legacy, default) |
| `RoundedOffsetShellType` | Use rounded corners when creating the shell |

### Complete Shell Feature Example

```python
import adsk.core, adsk.fusion, traceback

def run(context):
    ui = None
    try:
        app = adsk.core.Application.get()
        ui = app.userInterface

        doc = app.documents.add(adsk.core.DocumentTypes.FusionDesignDocumentType)
        design = adsk.fusion.Design.cast(app.activeProduct)
        rootComp = design.rootComponent
        features = rootComp.features

        # Create sketch circle on the XZ plane
        sketch = rootComp.sketches.add(rootComp.xZConstructionPlane)
        centerPoint = adsk.core.Point3D.create(0, 0, 0)
        sketch.sketchCurves.sketchCircles.addByCenterRadius(centerPoint, 10)

        # Get the profile and extrude a cylinder
        profile = sketch.profiles.item(0)
        distance = adsk.core.ValueInput.createByReal(2.5)
        extrudeFeature = features.extrudeFeatures.addSimple(
            profile, distance,
            adsk.fusion.FeatureOperations.NewBodyFeatureOperation
        )

        # Collect the end face to remove
        entities = adsk.core.ObjectCollection.create()
        entities.add(extrudeFeature.endFaces.item(0))

        # Create the shell feature input
        shellFeats = features.shellFeatures
        isTangentChain = False
        shellInput = shellFeats.createInput(entities, isTangentChain)
        shellInput.insideThickness = adsk.core.ValueInput.createByReal(0.5)
        shellInput.shellType = adsk.fusion.ShellTypes.SharpOffsetShellType

        # Create the shell feature
        shellFeature = shellFeats.add(shellInput)

    except:
        if ui:
            ui.messageBox('Failed:\n{}'.format(traceback.format_exc()))
```

## Mirror Features

Mirror features create a mirrored copy of bodies, features, faces, or components across a mirror plane.

Access via `rootComp.features.mirrorFeatures`

#### Methods

| Method | Returns | Description |
|--------|---------|-------------|
| `createInput(inputEntities, mirrorPlane)` | MirrorFeatureInput | Creates an input object for defining mirror parameters |
| `add(input)` | MirrorFeature | Creates a new mirror feature from a MirrorFeatureInput |
| `item(index)` | MirrorFeature | Returns the mirror feature at the given index |
| `itemByName(name)` | MirrorFeature | Returns the mirror feature with the given name |

#### Properties

| Property | Type | Description |

| Parameter | Type | Description |

|--------|-------------|
| `deleteMe()` | Delete the feature |

### Complete Mirror Feature Example

```python
import adsk.core, adsk.fusion, traceback

def run(context):
    ui = None
    try:
        app = adsk.core.Application.get()
        ui = app.userInterface

        doc = app.documents.add(adsk.core.DocumentTypes.FusionDesignDocumentType)
        design = adsk.fusion.Design.cast(app.activeProduct)
        rootComp = design.rootComponent
        features = rootComp.features

        # Create a rectangle sketch on the XZ plane
        sketch = rootComp.sketches.add(rootComp.xZConstructionPlane)
        startPoint = adsk.core.Point3D.create(0, 0, 0)
        endPoint = adsk.core.Point3D.create(5, 5, 0)
        sketch.sketchCurves.sketchLines.addTwoPointRectangle(startPoint, endPoint)

        # Extrude the rectangle
        prof = sketch.profiles.item(0)
        extInput = features.extrudeFeatures.createInput(
            prof, adsk.fusion.FeatureOperations.NewBodyFeatureOperation
        )
        distance = adsk.core.ValueInput.createByReal(5)
        extInput.setDistanceExtent(False, distance)
        ext = features.extrudeFeatures.add(extInput)

        # Get the body created by extrusion
        body = ext.bodies.item(0)

        # Create a construction plane offset from a face
        face = body.faces.item(0)
        planeInput = rootComp.constructionPlanes.createInput()
        offsetDistance = adsk.core.ValueInput.createByString('5 cm')
        planeInput.setByOffset(face, offsetDistance)
        plane = rootComp.constructionPlanes.add(planeInput)

        # Create input entities for mirror
        inputEntities = adsk.core.ObjectCollection.create()
        inputEntities.add(body)

        # Create the mirror feature
        mirrorFeats = features.mirrorFeatures
        mirrorInput = mirrorFeats.createInput(inputEntities, plane)
        mirrorFeature = mirrorFeats.add(mirrorInput)

    except:
        if ui:
            ui.messageBox('Failed:\n{}'.format(traceback.format_exc()))
```

| `item(index)` | RectangularPatternFeature | Returns the pattern feature at the given index |

### Complete Rectangular Pattern Example

```python
import adsk.core, adsk.fusion, traceback

def run(context):
    ui = None
    try:
        app = adsk.core.Application.get()
        ui = app.userInterface

        doc = app.documents.add(adsk.core.DocumentTypes.FusionDesignDocumentType)
        design = adsk.fusion.Design.cast(app.activeProduct)
        rootComp = design.rootComponent

        # Create a circle sketch on the XZ plane
        sketch = rootComp.sketches.add(rootComp.xZConstructionPlane)
        centerPoint = adsk.core.Point3D.create(0, 0, 0)
        sketch.sketchCurves.sketchCircles.addByCenterRadius(centerPoint, 3.0)

        # Extrude the circle into a cylinder
        prof = sketch.profiles.item(0)
        extInput = rootComp.features.extrudeFeatures.createInput(
            prof, adsk.fusion.FeatureOperations.NewBodyFeatureOperation
        )
        distance = adsk.core.ValueInput.createByReal(5)
        extInput.setDistanceExtent(False, distance)
        ext = rootComp.features.extrudeFeatures.add(extInput)

        # Get the body created by extrusion
        body = ext.bodies.item(0)

        # Create input entities for rectangular pattern
        inputEntities = adsk.core.ObjectCollection.create()
        inputEntities.add(body)

        # Get construction axis' for pattern directions
        xAxis = rootComp.xConstructionAxis
        yAxis = rootComp.yConstructionAxis

        # Define quantity and spacing
        quantityOne = adsk.core.ValueInput.createByString('3')
        distanceOne = adsk.core.ValueInput.createByString('8 cm')
        quantityTwo = adsk.core.ValueInput.createByString('3')
        distanceTwo = adsk.core.ValueInput.createByString('8 cm')

        # Create the rectangular pattern input
        rectPatterns = rootComp.features.rectangularPatternFeatures
        rectPatternInput = rectPatterns.createInput(
            inputEntities, xAxis,
            quantityOne, distanceOne,
            adsk.fusion.PatternDistanceType.SpacingPatternDistanceType
        )

        # Set the second direction
        rectPatternInput.setDirectionTwo(yAxis, quantityTwo, distanceTwo)

        # Create the rectangular pattern
        rectFeature = rectPatterns.add(rectPatternInput)

    except:
        if ui:
            ui.messageBox('Failed:\n{}'.format(traceback.format_exc()))
```

## Shared Enums

### PatternComputeOptions

Used by MirrorFeature, RectangularPatternFeature, and CircularPatternFeature when patterning features (not bodies).

| Value | Description |
|-------|-------------|
| `OptimizedPatternCompute` | Fastest compute option. May fail or produce incorrect results in rare cases (e.g., Split features) |
| `IdenticalPatternCompute` | Forces identical results in each pattern instance regardless of feature dependencies |
| `AdjustPatternCompute` | Slowest but most reliable. Almost never produces wrong results. Works correctly on Split features. This is the default |

### FeatureOperations Enum

Used when creating the initial bodies to shell, mirror, or pattern.

| Value | Description |
|-------|-------------|
| `NewBodyFeatureOperation` | Creates a new body |
| `JoinFeatureOperation` | Adds material to existing body |
| `CutFeatureOperation` | Removes material from existing body |
| `IntersectFeatureOperation` | Keeps only intersection |
| `NewComponentFeatureOperation` | Creates in new component |

## Common Patterns and Tips

### Editing Pattern Properties (Timeline Rollback)

When modifying properties of an existing pattern or mirror feature (e.g., changing the axis or mirror plane), the timeline must be rolled back to immediately before the feature:

```python
# Roll timeline back before the feature
feature.timelineObject.rollTo(True)

# Make changes
feature.axis = newAxis  # or feature.mirrorPlane = newPlane

# Roll timeline forward
design.timeline.moveToEnd()
```

### Input Entity Type Restriction

All four feature types enforce that `inputEntities` collections contain items of a single type. You cannot mix bodies and faces, or features and occurrences, in the same pattern/mirror operation.

```python
# CORRECT: all bodies
entities = adsk.core.ObjectCollection.create()
entities.add(body1)
entities.add(body2)

# INCORRECT: mixing types (will fail)
entities = adsk.core.ObjectCollection.create()
entities.add(body1)
entities.add(face1)  # Error: cannot mix bodies and faces
```

### Accessing Pattern Elements

Both pattern types and the mirror feature expose a `patternElements` collection on the result, providing access to each created instance:

```python
pattern = rectPatterns.add(rectPatternInput)
elements = pattern.patternElements
for i in range(elements.count):
    element = elements.item(i)
    # Each element provides access to its faces, bodies, etc.
```

### Suppressing Individual Pattern Elements

Rectangular and circular patterns support suppressing individual instances by their element IDs:

```python
# Get current suppressed elements
suppressed = rectFeature.suppressedElementsIds

# Suppress additional elements (by ID)
rectFeature.suppressedElementsIds = [2, 5]  # Suppress 2nd and 5th instances
```
